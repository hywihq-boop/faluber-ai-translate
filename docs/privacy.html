Faluber AI翻译 隐私政策

最后更新：2026年7月19日

概述

Faluber AI翻译（以下简称"本扩展"）高度重视用户隐私。本隐私政策说明我们如何收集、使用和保护您的信息。

我们收集的信息

  • API 配置信息：您提供的 API Key、API URL、模型名称等配置存储在浏览器的 chrome.storage.sync 中，仅用于调用您指定的 AI 翻译服务。
  • 翻译缓存：翻译结果缓存在 chrome.storage.local 中，用于避免重复翻译相同内容。缓存数据仅存储在本地，不会上传。
  • 使用统计：Token 用量统计（输入/输出 token 数）存储在 chrome.storage.local 中，仅用于在插件内显示用量信息。

我们不收集的信息

  • 浏览历史 — 本扩展不记录您访问的网页
  • 个人信息 — 本扩展不收集您的姓名、邮箱、IP 地址等任何个人信息
  • 网页内容 — 您浏览的网页内容仅在翻译时被发送到您指定的 AI API 服务商，本扩展本身不收集或存储网页内容
  • 任何分析或追踪数据 — 本扩展不包含任何第三方分析 SDK、埋点或广告追踪

数据如何存储

所有数据（API 配置、翻译缓存、Token 统计）均存储在浏览器本地存储中：

  • chrome.storage.sync — API 配置（跨设备同步）
  • chrome.storage.local — 翻译缓存和使用统计（仅本机）

这些数据不会发送到本扩展的开发者服务器。本扩展没有自己的后端服务器。

数据如何共享

翻译时，需要翻译的文本会发送到您自己指定的 AI API 服务商（如 DeepSeek、OpenAI 等）。数据的传输和处理受该服务商的隐私政策约束。本扩展开发者无法访问这些数据。

您的控制权

  • 您可以随时在插件 Popup 中删除或更换 API 配置
  • 您可以随时在悬浮球详情面板中清除翻译缓存
  • 卸载扩展将自动清除所有本地存储的数据

第三方服务

本扩展通过您配置的 API 服务商进行翻译。请参阅您使用的 API 服务商的隐私政策：

  • DeepSeek
  • OpenAI
  • Groq
